Convolutional Neural Networks

Masters of image and spatial data

‚ùå The Dense Layer Problem

Why Dense Layers Struggle with Images

  • Lose spatial structure: Flatten 2D image to 1D vector
  • Too many parameters: 28√ó28 image = 784 inputs per neuron
  • No translation invariance: Cat in top-left ‚â† cat in bottom-right
  • Ignore local patterns: Don't understand edges, textures

Real Example

A 224√ó224 color image (150,528 pixels) ‚Üí Dense layer = 150 million parameters for just one hidden layer!

‚úÖ The CNN Solution

How CNNs Fix These Problems

  • Preserve spatial structure: Work with 2D feature maps
  • Parameter sharing: Same filter across entire image
  • Translation invariant: Detect patterns anywhere
  • Hierarchical features: Edges ‚Üí shapes ‚Üí objects

Parameter Efficiency

Same 224√ó224 image ‚Üí CNN with 32 filters = only 896 parameters (99.4% reduction!)

üß± Essential CNN Layers

Two Key Building Blocks

Convolutional Layers:


layers.Conv2D(
    filters=32,
    kernel_size=3,
    activation='relu'
)

Purpose: Detect features like edges and textures

Pooling Layers:


layers.MaxPooling2D(
    pool_size=2,
    strides=2
)

Purpose: Reduce size, add translation invariance

The Pattern

Convolution ‚Üí Pooling ‚Üí Repeat ‚Üí Flatten ‚Üí Dense

üèóÔ∏è Typical CNN Architecture

Complete CNN in TensorFlow


model = keras.Sequential([
    layers.Conv2D(32, 3, activation='relu', input_shape=(28, 28, 1)),
    layers.MaxPooling2D(2),
    layers.Conv2D(64, 3, activation='relu'),
    layers.MaxPooling2D(2),
    layers.Flatten(),
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')
])

What This Architecture Does

‚Ä¢ 32 filters (3√ó3) ‚Üí detect basic features
‚Ä¢ Pool ‚Üí reduce size
‚Ä¢ 64 filters (3√ó3) ‚Üí detect complex features
‚Ä¢ Pool ‚Üí reduce size again
‚Ä¢ Flatten & Dense ‚Üí final classification

Recurrent Neural Networks

Masters of sequences and time

‚è∞ Understanding Sequential Data

Where Order Matters

  • Text: "I love deep learning" vs "Deep learning love I"
  • Time series: Stock prices, weather, sensor data
  • Audio: Music, speech recognition
  • Video: Action recognition, temporal patterns

Why Dense/CNN Networks Fail

  • Fixed input size: Can't handle variable-length sequences
  • No memory: Each input processed independently
  • Position matters: Same word means different things in different contexts

üß† RNN Types in TensorFlow

Simple RNN (Basic Memory)


layers.SimpleRNN(64, return_sequences=True)

Good for: Short sequences, simple patterns

LSTM (Long Short-Term Memory)


layers.LSTM(64, return_sequences=True)

Good for: Long sequences, complex dependencies

GRU (Gated Recurrent Unit)


layers.GRU(64, return_sequences=True)

Good for: Faster than LSTM, similar performance

Transfer Learning

Standing on the shoulders of giants

üò∞ Training From Scratch Problems

Why Starting From Scratch Is Hard

  • Time: Weeks or months of training time
  • Data: Need millions of labeled examples
  • Compute: Expensive GPUs and clusters
  • Expertise: Lots of hyperparameter tuning
  • Failure: High risk of poor results

Reality Check

ImageNet classification: 1.2M images, 1000 classes, 100+ GPU hours, $1000s in compute costs

üöÄ Transfer Learning Solution

Stand on the Shoulders of Giants

  • Speed: Hours instead of weeks
  • Data: Work with hundreds of examples
  • Cost: Run on single GPU
  • Results: Often better than training from scratch
  • Reliability: Proven architectures that work

The Insight

Pre-trained models already learned to detect edges, shapes, textures - reuse this knowledge!

üîí Feature Extraction Approach

Freeze Pre-trained Layers


base_model = ResNet50(
    weights='imagenet',
    include_top=False
)
base_model.trainable = False  # Freeze all layers

model = keras.Sequential([
    base_model,
    layers.GlobalAveragePooling2D(),
    layers.Dense(10, activation='softmax')
])

When to Use

‚Ä¢ Small dataset (< 1000 images)
‚Ä¢ Similar to ImageNet (photos of objects)
‚Ä¢ Fast training - only train final layer

‚öôÔ∏è Fine-tuning Approach

Unfreeze Some Pre-trained Layers


base_model = ResNet50(
    weights='imagenet',
    include_top=False
)
base_model.trainable = True

# Freeze early layers, unfreeze later ones
for layer in base_model.layers[:-20]:
    layer.trainable = False

When to Use

‚Ä¢ Larger dataset (> 1000 images)
‚Ä¢ Different from ImageNet (medical images, etc.)
‚Ä¢ Maximum accuracy - adapt features to your data
